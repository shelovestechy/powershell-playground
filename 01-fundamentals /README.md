# 01 - Fundamentals

Tämä osio keskittyy PowerShellin ytimeen: objekteihin. Tavoitteena on oppia pois tekstipohjaisesta ajattelusta.

### Oppimistavoitteet:
- [ ] **The Pipeline:** Ymmärrä miten `|` siirtää kokonaisia objekteja komennolta toiselle.
- [ ] **Object Anatomy:** Opi käyttämään `Get-Member` -komentoa löytääksesi metodit ja ominaisuudet.
- [ ] **Variables & Data Types:** Hallitse muuttujat, taulukot (Arrays) ja sanakirjat (Hashtables).
- [ ] **Filtering & Sorting:** Käytä `Where-Object` ja `Sort-Object` suodattamiseen.
- [ ] **The Help System:** Opi käyttämään `Get-Help -Examples` ja `Get-Command`.

### Harjoitukset:
*Luo tänne .ps1 tiedostoja, joissa kokeilet yllä olevia asioita.*
