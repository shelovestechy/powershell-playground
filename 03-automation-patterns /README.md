# 03 - Advanced Automation & Toolmaking

Täällä rakennetaan työkaluja, jotka on tarkoitettu muidenkin kuin itsesi käytettäväksi.

### Oppimistavoitteet:
- [ ] **Advanced Functions:** Lisää `[CmdletBinding()]` tehden funktiosta oikean cmdletin.
- [ ] **Modules:** Paketoi funktiot `.psm1` moduuliksi.
- [ ] **Logging:** Kirjoita lokia tiedostoon tai käytä `Write-Verbose` debuggaamiseen.
- [ ] **Performance:** Testaa koodin nopeutta `Measure-Command` avulla.
- [ ] **API basics:** Ensimmäiset kokeilut `Invoke-RestMethod` kanssa.
