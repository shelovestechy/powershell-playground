# 04 - IAM & Cloud Identity

Senior-tason osaaminen: Identiteetin hallinta, pilviympäristöt ja tietoturva.

### Oppimistavoitteet:
- [ ] **Microsoft Graph SDK:** Miten hallitaan M365-käyttäjiä ohjelmallisesti.
- [ ] **Active Directory:** Käyttäjien ja ryhmien hallinta hybridiympäristössä.
- [ ] **Entra ID:** Automaatiot Joiner-Mover-Leaver prosesseihin.
- [ ] **RBAC:** Oikeuksien tarkistaminen ja delegointi.
- [ ] **Auditing:** Raportointi siitä, kuka teki ja mitä.

### IAM-Projektit:
*Esimerkiksi: "Käyttäjien lisenssi-auditoija" tai "AD-ryhmäjäsenyys raportti".*
