# 02 - Logic & Scripting Basics

Tässä osiossa opetellaan tekemään päätöksiä ja toistamaan toimintoja älykkäästi.

### Oppimistavoitteet:
- [ ] **Control Flow:** Rakennat logiikkaa `if`, `else` ja `switch` rakenteilla.
- [ ] **Loops:** Toista asioita `foreach`, `while` ja `do-until` silmukoilla.
- [ ] **Error Handling:** Käytä `Try/Catch` blokkeja hallittuun virheen käsittelyyn.
- [ ] **Functions:** Tee koodista uusiokäytettävää funktioiden avulla.
- [ ] **File I/O:** Lue ja kirjoita CSV- ja JSON-tiedostoja (`Import-Csv`, `ConvertTo-Json`).
- [ ] **Scopes:** Ymmärrä missä muuttuja on elossa (Global vs. Local).
